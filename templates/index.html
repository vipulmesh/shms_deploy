About This System

This Smart Community Health Monitoring System is designed to help rural health workers track and predict water-borne disease outbreaks. The system uses a rule-based AI algorithm to assess risk levels based on disease cases and environmental factors like rainfall.

Technology Stack:

  • Frontend: HTML5, CSS3, JavaScript
  • Backend: Python Flask
  • Database: SQLite

Features:

  • Manual data entry by health workers
  • Rule-based risk prediction
  • Visual dashboard with alerts
